# 趣搭星球 - 后端模块说明
## 一、模块功能
### 核心定位
作为“趣搭星球”响应式Web应用的服务端核心，提供用户认证、地理位置管理、搭子推荐、活动日记管理、文件上传等核心能力，支撑前端所有业务场景的API调用，重点实现基于Redis Geo的高性能“附近的人”搭子推荐功能。

### 核心子模块功能
| 子模块         | 功能描述                                                                 |
|----------------|--------------------------------------------------------------------------|
| 用户与认证模块 | 负责用户注册、登录（JWT认证）、用户信息查询/修改、用户地理位置上报          |
| 搭子推荐模块   | 基于Redis Geo实现附近用户筛选，结合兴趣标签计算推荐分数，返回个性化搭子列表 |
| 活动日记模块   | 活动创建、查询、详情展示，共享日记的创建、编辑、查询                      |
| 文件上传模块   | 接收前端图片上传请求，保存到服务器本地并返回可访问URL                     |

## 二、技术选型
| 技术/框架 | 版本       | 选型理由                                                               |
|------|------------|------------------------------------------------------------------------|
| Spring Boot | 3.2.x      | 主流轻量级Java Web框架，自动配置、快速开发，降低后端开发复杂度           |
| MySQL | 8.0        | 成熟稳定的关系型数据库，适合存储用户、活动、日记等结构化业务数据           |
| MyBatis | 3.5.x      | 基于MyBatis的ORM框架，简化CRUD操作，支持分页、条件构造器等实用功能     |
| Redis | 7.0.x      | 高性能内存数据库，通过Geo类型实现“附近的人”查询，同时可用于缓存热点数据     |
| JWT  | 0.11.5     | 轻量级无状态认证方案，无需服务端存储会话，适配前后端分离架构               |

## 四、运行方式
### 前置条件
1. 安装JDK 17+（Spring Boot 3.x要求）；
2. 安装MySQL 8.0，创建数据库，执行初始化SQL（创建用户、活动等表）；
3. 安装Redis 7.0+，确保Redis服务启动且无密码（或修改application.yml配置密码）；
4. 配置application.yml中的数据库连接、Redis地址、图片存储路径等参数。

### 启动方式
#### 方式1：IDE启动
1. 打开IntelliJ IDEA，导入backend目录下的Maven项目；
2. 等待Maven依赖下载完成；
3. 运行`FunMatePlanetApplication.java`的main方法；
4. 控制台输出`Started FunMatePlanetApplication in x.xx seconds`表示启动成功。

#### 方式2：Maven命令启动
1. 进入backend目录；
2. 执行打包命令：`mvn clean package -DskipTests`；
3. 执行启动命令：`java -jar target/FunMatePlanet-1.0-SNAPSHOT.jar`；
4. 访问`http://localhost:8080/api/users/me`验证服务是否可用（需先登录获取JWT）。
