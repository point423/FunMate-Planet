## 核心页面

### 1. Home — 主页（匹配伙伴）

极简全屏布局：

- 两个主操作按钮：**自动匹配伙伴**（荧光绿填充）、**浏览附近伙伴列表**（白色描边）
- 自动匹配：系统推送一位用户，展示其主页信息（活动数、评分、排名、标签、公开日记），底部可选**申请好友 Apply** 或 **跳过 Skip**
- 浏览列表：跳转至 Activity > 寻找伙伴

### 2. Chat — 聊天界面

三栏式布局：

- 左栏（导航）

  ：切换 Partners / Applications 两个 Tab

  - Partners：伙伴头像列表，点击激活中栏聊天
  - Applications：收到的申请 + 我发出的申请，点击可在右栏查看完整主页，底部接受 / 拒绝

- **中栏（聊天）**：历史消息气泡 + 输入框（含表情、更多功能展开面板）

- **右栏（活动）**：日历组件（展示活动时间轴）+ 活动详情卡片（名称·地点·参与人·起止时间）+ 结束活动按钮

### 3.  journal—活动日记

两栏布局：

- **左半**：日记卡片列表（名称·参与人·起始日期）+ 新建日记按钮（荧光绿）
- **右半**：选中日记详情 — 活动照片轮播 · 日记文本 · 评价伙伴提示 · 查看参与人/位置/结束时间的弹窗按钮

### 4. partner—寻找伙伴

- **顶部统计栏**：我的活动数 · 评分 · 排名（荧光绿高亮数字）
- **右侧排行榜**：每周好评度 Top 榜（头像·名称·分数）
- **主区域**：附近伙伴推荐列表，支持标签多选筛选；点击伙伴卡片进入其主页
- **伙伴主页**：展示活动数·评分·排名·公开日记·最近活动，底部 **Apply（申请好友）** / **Discard（放弃）** 按钮，点击后均自动返回上一页

### 5. My — 我的主页

布局与伙伴主页一致，差异：

- 无 Apply / Discard 按钮
- 多一个 **Edit（编辑资料）** 按钮，可修改头像·昵称·标签·简介

## 交互说明

| 操作                                                | 行为                                                         |
| --------------------------------------------------- | ------------------------------------------------------------ |
| 登录成功（新用户）                                  | 跳转至标签选择页，完成后进入 Home                            |
| 登录成功（老用户）                                  | 直接进入 Home                                                |
| Home · 点击「自动匹配伙伴auto match」               | 弹出随机推荐用户主页，Apply → 发送申请并跳回 Home；Skip → 下一个推荐 |
| Home · 点击「浏览附近伙伴列表scan list」            | 跳转至 Activity > 寻找伙伴                                   |
| Chat · 点击伙伴                                     | 中栏展示聊天记录，右栏展示活动信息                           |
| Chat · 聊天框点击「+」                              | 展开浮层菜单：发起活动 · 发送定位 · …                        |
| Chat · Applications · 接受申请                      | 申请人移入 Partners 列表，右栏变为聊天状态                   |
| Chat · Applications · 拒绝申请                      | 申请从列表移除                                               |
| Activity · 点击日记卡片                             | 右半渲染日记详情                                             |
| Activity · 新建日记                                 | 弹出创建表单                                                 |
| Activity · 日记详情弹窗按钮（参与人 / 位置 / 时间） | 弹出对应信息弹窗                                             |
| Activity · 伙伴partner卡片 Apply                    | 向该用户发送申请，跳回寻找伙伴列表                           |
| Activity · 伙伴partner卡片 Discard                  | 直接跳回寻找伙伴列表                                         |
| Chat · 结束活动                                     | 弹出确认弹窗，确认后活动归档至日记，触发「评价伙伴」提示     |
| 标签筛选（寻找伙伴）                                | 多选标签，实时过滤伙伴列表                                   |
| 排行榜                                              | 每周一 00:00 自动刷新，展示本周好评度 Top 榜                 |
