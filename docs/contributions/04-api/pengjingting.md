## 遇到的问题和解决
1. **问题**：Redis 地理位置搜索报 `Error in execution`，原因是 JSON 序列化导致存入 ID 带引号，造成解析冲突。
   **解决**：将地理位置操作切换为 `StringRedisTemplate`，确保 ID 存取为纯文本格式，解决了空间计算异常。
2. **问题**：自动化测试脚本在执行过程中频繁出现 404 或 405 错误。
   **解决**：通过补全 Controller 缺失的接口方法，并将请求方式统一为标准 JSON Body 传参，确保了脚本与接口协议 100% 对齐。
