## 我完成的工作
本次项目中我主要负责**后端开发**工作，在 UI 设计环节主要参与前期需求沟通与方案确定，具体工作如下：
1. **视觉风格与配色协同**
   与团队前端成员共同讨论、确定项目整体主题风格，一起敲定项目核心配色方案，为后续 UI 视觉设计奠定基础。
2. **界面与功能需求梳理**
   结合后端业务逻辑与项目整体需求，和同伴共同讨论、梳理需要实现的界面内容、页面模块及核心功能，明确 UI 设计的范围与落地方向。
3. **前后端设计协同对接**
   配合前端成员的 UI 设计工作，同步思考后续后端功能逻辑，为界面内容与功能规划提供业务层面的支持。
