## 设计思路说明

本次设计在 Slowly 交友软件（温暖手账风）与 Soul 社交软件（沉浸式暗色星球）的视觉语言基础上进行融合创新，最终确定为：极简暗色基调 × 荧光绿单色强调的视觉风格

核心设计决策：

1. 黑底白字：高对比度，符合 Web 端长时间浏览的易读性要求
2. 荧光绿 `#95FF8D`：唯一彩色，作为所有主操作按钮、激活状态、重要数据的视觉锚点，形成强烈记忆点
3. 三栏式 Chat 布局：参考 Slack / Discord 的左侧导航 + 中部内容 + 右侧上下文面板结构，适合 Web 端宽屏使用
4. Activity 竖向导航：将日记和搭子发现整合进同一界面，减少页面跳转层级

## 遇到的问题和解决

1. **问题**：黑白双色方案容易显得单调，区分度不足。 

   **解决**：引入荧光绿 `#95FF8D` 作为唯一强调色，并建立严格的使用规则（只用于主操作按钮、激活状态、重要数字），避免滥用导致失去强调作用。

2. **问题**：Chat 界面信息密度高（伙伴列表 + 聊天 + 活动日历），布局容易混乱。 

   **解决**：采用三栏固定布局，左栏宽度固定（导航）、中栏自适应（聊天内容主区域）、右栏固定（活动上下文），参考 Discord 布局逻辑，各区域职责清晰。
