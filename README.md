## 项目简介
趣搭星球是一款移动端的响应式Web应用。核心功能是兴趣标签匹配共同爱好的搭子，搭配Redis Geo实现的高性能地理位置查询计算能力，从而帮大家高效找到身边同频的搭子。并且可以一键发起线下兴趣活动，还有图文日记功能，能和同行的搭子一起记录线下相聚的点滴时光。

我们的目标用户聚焦有线下兴趣社交需求的年轻群体，主要是学生党和年轻人，核心就是满足大家找同好、拓展舒服的社交圈、丰富日常休闲生活的需求，拒绝无效尬聊，帮大家精准匹配到合拍的搭子。


### 核心后端功能 (已 100% 完成 API 覆盖)
- **认证系统**：支持 Mock JWT 登录/注册/登出全闭环。
- **地理发现**：基于 **Redis Geo** 实现毫秒级的“附近的人”搜索。
- **社交网络**：支持单向关注、粉丝统计以及双向好友申请握手逻辑。
- **活动闭环**：发起活动、加入活动、活动状态管理。
- **评价体系**：活动后用户互评，系统自动计算并更新**好评排行榜**。
- **活动日记**：支持多图展示的线下活动回忆墙。


## 技术栈（初步规划）
- **前端**：Vue 3 + Vite，Element Plus，Vue Router，Pinia，Axios
- **后端**：Spring Boot,Spring Data JPA，JWT 
- **数据库**：MySQL 8.0，Redis(Geo)
- **部署**：Docker + Docker Compose 



## 运行与测试指南

### 1. 快速启动 (Docker 环境)
在 Linux 服务器上执行：
```
# 1. 编译打包 (在 backend 目录下)
mvn clean package -DskipTests

# 2. 一键启动全家桶 (MySQL, Redis, Backend)
docker compose up -d --build
```
